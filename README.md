# CDMX Retro Planner

Single-page app with three planning sections:

- `Trending Restaurants`: Mexico City recommendations from The Infatuation, Eater, and The New York Times 36 Hours guide
- `Museums`: curated Mexico City museum/gallery list
- `Attractions`: Atlas Obscura attractions in Mexico City

## Filtering

Restaurant list can be filtered by:

- source
- time of day open (breakfast/lunch/dinner/late-night)
- price category (`$` to `$$$$`)
- neighborhood

Restaurant cards include street addresses for each listed venue.

## Google Maps

- Each restaurant, museum, and attraction card includes an `Open in Google Maps` link.
- Use `Download All Spots CSV` and import that file in Google My Maps to see all spots labeled on one map.
